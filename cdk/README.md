# Welcome to your CDK TypeScript project

This is a blank project for CDK development with TypeScript.

The `cdk.json` file tells the CDK Toolkit how to execute your app.

## Useful commands

* `bun run build`   compile typescript to js
* `bun run watch`   watch for changes and compile
* `bun run test`    perform the jest unit tests
* `bun cdk run deploy '*' `  deploy this stack to your default AWS account/region
* `bun cdk run destroy '*' `
* `bun cdk run diff`    compare deployed stack with current state
* `bun cdk run synth`   emits the synthesized CloudFormation template
